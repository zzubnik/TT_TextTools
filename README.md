# TT_TextTools
Custom text tools for prompt generation in ComfyUI.

Story Combiner - For generating multiple prompts from sets of words. Lets you create a lot of variation in yourt prompts. The following screenshots are the same text(s), just randomised by the node.

TextInput - a multi-line text box for feeding Story Combiner with text(s)

TextFileSelectorNode - Will select a random line from a user specified .txt file
